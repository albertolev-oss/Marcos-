# Cadera 3D · Artroplastia Total

Sitio educativo interactivo sobre artroplastia total de cadera.

## Link esperado en GitHub Pages

https://albertolev-oss.github.io/Marcos-/hip-arthroplasty-3d/

## Archivos

- `index.html`
- `styles.css`
- `app.js`

## Funciones

- Modelo 3D interactivo con Three.js.
- Controles táctiles para iPhone.
- Botón `Probar JS` para confirmar que JavaScript funciona.
- Panel con inclinación del cotilo, anteversión y offset femoral.
- Modos de vista: frente, lateral y axial.

## Nota médica

Modelo didáctico aproximado. No sirve para planificación quirúrgica real, navegación intraoperatoria ni medición radiográfica exacta.
