# Cadera 3D Pro · Artroplastia Total

Modelo educativo interactivo de artroplastia total de cadera para GitHub Pages.

## Link público

https://albertolev-oss.github.io/Marcos-/hip-arthroplasty-3d/

## Mejoras incluidas

- Interfaz móvil más limpia.
- Modelo 3D procedural más completo.
- Pelvis, acetábulo, fémur proximal, cotilo, liner, cabeza, cono modular y vástago.
- Etiquetas flotantes.
- Modo Rx.
- Vistas AP, lateral, axial y oblicua.
- Modo explotado.
- Movimiento educativo.
- Panel con inclinación, anteversión, offset, longitud modular y opacidad ósea.
- Botón `Probar JS`.

## Archivos

- `index.html`
- `styles.css`
- `app.js`

## Nota médica

Modelo didáctico aproximado. No sirve para planificación quirúrgica real, navegación intraoperatoria ni medición radiográfica exacta.
